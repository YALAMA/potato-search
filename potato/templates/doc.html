谷歌搜索 API:
调用方式: /api/search/?q={0}&page={1}&key={2}
{0} 替换为查询字符, 类型为 str
{1} 替换为第一条结果的索引, 第一页第一个为 1, 每次固定返回 10 个, 所以第二页第一个为 11, 类型为 int
{2} [可选参数] 替换为自己的 API KEY, 类型为 str
例: /api/search/?q=python&page=1
429 表示今日请求次数已用尽, 请换用其他 API KEY
400 表示 API KEY 错误

维基百科 API:
调用方式: /api/wiki/?q={0}
{0} 替换为词条, 类型为 str
例: /api/wiki/?q=上海
